Projects

  • TicketManager

    An online platform to distribute tickets for an event. Each ticket consist of a unique token wich can be scanned and validated by the organizer of the event.

    The client app uses ReactJs + Semantic UI + apollo-client , and speaks to a GraphQL server, which handles user authentication as well as other operations, storing the data in a MongoDB database.

  • Drinks & Waffles

    A progressive web application to register orders from a family restaurant, built with BootstrapVue + Axios , connected to a REST API, which was developed in TypeScript, using Express and Mongoose.

  • Maka'i

    A multiplayer browser game based on a traditional card game from Paraguay. The server was built with TypeScript and Socket.io, while the client uses Vue.js and Bootstrap 5.
